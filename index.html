xmledit by sukima

A filetype plugin for VIM to help edit XML files

XML Edit is a file type plugin to help edit XML documents. It includes tag completion and tag jumping.

Dependencies

VIM 7.0

License

Expand or copy these directories to your VIM home. On Unix systems this is `~/.vim` On MS-DOS it is your `$VIM/vimfiles` folder. On a Unix system, if you wish to make this script avaliable system wide place this package into your $VIMRUNTIME directory. The documentation will be installed automatically the first time you open or edit an XML file. To do so try `:e test.xml` the first time and it will install the documentation to `/doc` Now read doc/xml-plugin.txt or type `:help xml-plugin.txt` in VIM.
